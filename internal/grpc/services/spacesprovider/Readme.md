Spaces Provider
===============

The spaces provider is based on storage spaces. It provides a file tree of human readable aliases for all spaces the current user has access to:

```
├── personal
│   └── einstein
├── projects
│   └── atlas
└── shares
    ├── othershare
    └── someshare
```
